This project is an end-to-end Data Science application that predicts mobile phone market prices based on technical specifications (RAM, Storage, Battery, etc.) using Machine Learning algorithms and serves the model via a dynamic Streamlit web application.

## Overview
- **Problem:** Uncertainty and lack of standardized pricing in the second-hand and retail mobile phone market.
- **Solution:** Feature-based price prediction using Machine Learning regression models.
- **Business Value:** Helps buyers and sellers determine the optimum market price instantly, ensuring fair trade and maximizing profitability.

## Tech Stack
- **Data Processing & Analysis:** Pandas, NumPy
- **Machine Learning:** Scikit-Learn
- **Deployment:** Streamlit Cloud
